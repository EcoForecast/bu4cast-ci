Bucket Organization:
--------------------

Read:
- Most everything is in bu4cast-ci-read/challenges/project_id=bu4cast
- Drivers contains ...?
- Metadata contains field site IDs and units
- Students contains any files the students may want (beyond data)
- Targets contains the current downloaded data for each topic that can then be used to predict future values

Write:
- Most everything is in bu4cast-ci-write/challenges/project_id=bu4cast
- Parquet and Processed_Submissions are created during process_submissions.R
- Raw_submissions is where all submission files are moved after being successfully processed
- Submissions should only contain unprocessed and failed processed files (THIS IS THE ONLY PLACE STUDENTS SHOULD BE SUBMITTING)

How to find Secret and Access Keys:
-----------------------------------

Project Name: software-application-innovation-lab-sail-projects-fcd6dfa
Link: https://rhods-dashboard-redhat-ods-applications.apps.shift.nerc.mghpcc.org/projects/software-application-innovation-lab-sail-projects-fcd6dfa?section=connections

Both bu4cast-ci-read and bu4cast-ci-write have the same secret and access keys. To copy them, go to the three dots next to either and click edit. From there, you can scroll down and copy them. 
These are also the login credentials to the Minio console (https://minio-console.apps.shift.nerc.mghpcc.org/browser/bu4cast-ci-read). The Access Key is the Username and the Secret Key is the password.

How to Set Public Read/Write Access:
------------------------------------

Use these terminal commands (this is all based on Mac, but I assume Linux and Windows are similar).
Replace ACCESS_KEY and SECRET_KEY with their respective values from "How to find Secret and Access Keys"

Read:

brew install minio/stable/mc
mc alias set myminio https://minio-s3.apps.shift.nerc.mghpcc.org ACCESS_KEY SECRET_KEY
mc anonymous set download myminio/bu4cast-ci-read

Write:

brew install minio/stable/mc
mc alias set myminio https://minio-s3.apps.shift.nerc.mghpcc.org ACCESS_KEY SECRET_KEY
mc anonymous set upload myminio/bu4cast-ci-write
